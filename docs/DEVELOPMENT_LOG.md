# AeroSim 研发日志 (Research & Development Log)

## 2026-02-26 - 项目重启与架构升级
### 1. 核心决策
- **3D 建模驱动**: 放弃解析公式，转为基于 STL 几何模型的气动特性计算。
- **GPU 求解器**: 开发基于 CUDA 的表面元/改进牛顿法求解器，利用本机 GPU 硬件加速。
- **自动调参**: 确认所有气动和弹道设计参数必须进入 `Config` 结构体，严禁硬编码。使用算法（如差分进化）进行全局参数寻优。

### 2. 架构规范
- **配置化**: 每一项设计参数（包括 3D 模型生成的长宽比、后掠角等）必须在 `HGV1Config` 或其子结构中定义。
- **日志维护**: 本文件将同步维护所有重大变更和参数调优记录，遵循“只增不减”原则。
