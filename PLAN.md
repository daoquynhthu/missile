# 高保真弹道仿真工程计划书 (High-Fidelity Ballistic Simulation Project)

## 1. 项目目标
构建一套基于 C++ 的全保真、高精度弹道仿真引擎，专门用于高超音速飞行器（如钱学森弹道）的动力学特性研究、热环境分析及关键参数反推。

## 2. 核心技术架构

### 2.1 地球物理模型 (Geophysical Models)
- **引力场**：实现 EGM2008 (Earth Gravitational Model 2008)，通过球谐函数展开（Spherical Harmonics）计算全球非均匀引力加速度。
- **椭球体**：采用 WGS-84 标准，精确计算大地高度及坐标转换（ECI <-> ECEF <-> NED）。
- **地球自转**：考虑科氏力 (Coriolis) 与离心加速度。

### 2.2 大气环境模型 (Atmospheric Models)
- **高空/中低空**：集成 NRLMSISE-00 大气模型，考虑太阳活动（F10.7）、地磁指数及季节性密度波动。

### 2.3 动力学求解器 (Dynamics Solver)
- **自由度**：全流程 6-DOF（六自由度）方程，包含平动与转动耦合、变质量效应、动量矩守恒。
- **数值积分**：采用自适应步长的 Runge-Kutta-Fehlberg 8(9) 算法 (RKF89) 或 Dormand-Prince (DP853)，确保长时间积分的能量守恒与精度。

### 2.4 气动与热防护 (Aero-thermodynamics)
- **气动数据库**：支持基于马赫数、攻角、侧滑角的多维线性插值。
- **热流计算**：集成 Fay-Riddell 驻点热流公式与辐射散热平衡计算。

## 3. 开发阶段规划

### 第一阶段：基础设施与物理内核 (Week 1)
- [ ] 初始化 Git 仓库与 CMake 工程。
- [ ] 实现基础坐标系转换类。
- [ ] 编写 EGM2008 引力场计算模块（需预加载球谐系数表）。

### 第二阶段：环境与动力学集成 (Week 2)
- [ ] 集成 NRLMSISE-00 C 库封装。
- [ ] 构建 6-DOF 状态空间方程。
- [ ] 实现变质量推力矢量控制模型。

### 第三阶段：高精度求解与验证 (Week 3)
- [ ] 开发 RKF89 自适应步长求解器。
- [ ] 实现平衡滑翔 (EGC) 指令制导算法。
- [ ] 与公开基准弹道进行精度比对。

### 第四阶段：并行加速与优化 (Week 4+)
- [ ] **CUDA 迁移**：将引力场计算与大规模蒙特卡洛参数扫描迁移至 GPU。
- [ ] 实现基于模板的逆向参数寻优算法。

## 4. 精度保证体系
- 所有的位置计算采用 `double`（64位浮点），关键累积量考虑使用双-双精度（Double-Double）。
- 建立能量守恒检验机制，实时监控积分误差。
- 定期进行单位测试与坐标系变换的闭环验证。
