# 项目交手文档 (Handoff Documentation)

**最后更新日期**: 2026-02-26
**当前版本**: v0.5 (钱学森弹道跳跃滑翔验证阶段)

致后继开发者：

本项目是一个高保真度的导弹 6自由度 (6-DOF) 仿真平台，专注于高超声速滑翔飞行器 (HGV) 的**钱学森弹道 (Skip-Glide)** 模拟与参数优化。

目前已完成**核心物理引擎**、**制导律框架**以及**自动化参数调优**闭环。

---

## 1. 核心架构与进展

### 1.1 物理引擎 (`AeroSim` Core)
- **重力模型**: EGM2008 (4阶/12阶可选)，支持 J2/J3 项修正。
- **大气模型**: 集成 NRLMSISE-00 (Fortran -> C++ 接口)，提供高精度高层大气密度/温度数据。
- **动力学**: RK4 积分器，支持变质量刚体动力学。
- **气动模型**: 基于牛顿撞击理论 (Newtonian Impact Theory) 的高超声速气动估算 (Waverider Configuration)。
  - *最新更新*: 修正了高攻角下的升阻比特性，调整了压心 (CP) 位置以保证静稳定裕度 (Static Margin) 约为 4-5%。

### 1.2 制导与控制 (`GNC` Module)
制导律位于 `include/gnc/guidance.hpp`，实现了分段制导逻辑：
1.  **BOOST (助推段)**: 程序角 (Pitch Program) + 目标方位角对准。
    - *优化*: 实现了基于目标坐标的发射方位角自动计算。
2.  **COAST (惯性飞行段)**: 零攻角弹道飞行，直到再入界面 (H=50km)。
3.  **GLIDE (滑翔段)**: **核心难点**。
    - **QEGG (准平衡滑翔)**: 通过高度/升沉率反馈控制攻角 (AoA)。
    - **Reentry Pull-up (再入拉起)**: *新增逻辑*。当检测到低空 (<60km) 且大下沉率 (<-50m/s) 时，强制最大攻角 (20°+) 进行“跳跃”，防止坠毁。
4.  **TERMINAL (末端)**: 比例导引 (PN)，目前处于初步实现阶段。

### 1.3 仿真与优化工具链
- **主程序**: `src/main.cpp` 支持命令行参数输入，便于批处理运行。
- **优化脚本**: `tune_parameters.py` (Python)。
  - **算法**: 从 Nelder-Mead 升级为 **差分进化 (Differential Evolution)**，以应对非凸、多峰的参数空间。
  - **目标函数**: 最大化射程 (Range) + 严格限制远地点 (<100km) + 防止坠毁。
  - **优化变量**: [助推开始时间, 助推俯仰速率, 滑翔攻角, 拉起高度, 拉起攻角]。

## 2. 当前状态与已知问题

### 2.1 运行状态
- **编译**: `cmake --build build --config Release`
- **测试**: 运行 `python tune_parameters.py` 即可启动自动化调参。
- **地标**: 发射点设为酒泉 (Jiuquan)，目标点设为太平洋预定海域。

### 2.2 关键问题 (Critical Issues)
1.  **射程不足**: 目前仿真射程约 1300km，距离目标 (4000km+) 仍有差距。
    - *原因推测*: 气动模型的升阻比 (L/D) 在高超声速下可能偏低；助推段能量管理需进一步优化（是否需要两级助推？）。
2.  **“海豚跳”振荡**: 在滑翔段可能出现剧烈的高度振荡。需精细调节 `guidance.hpp` 中的 PID 增益 (`kp_alt`, `kp_vz`)。
3.  **热防护未建模**: 当前未考虑气动加热约束，激进的拉起机动可能导致烧蚀。

## 3. 后续开发指南 (Roadmap)

### 3.1 短期任务 (Priority: High)
1.  **精细化气动数据**: 替换 `missile_config.cpp` 中的解析公式，使用 CFD 计算生成的真实气动数据表 (CSV/JSON 导入)。
2.  **解决射程问题**: 
    - 检查阻力系数 (Cd) 是否过大。
    - 尝试优化助推器关机点参数。
3.  **完善末端制导**: 确保在滑翔末段 (H<20km) 能平滑交班给 PN 制导律，实现精确打击。

### 3.2 长期任务
1.  **热环境仿真**: 引入 Fay-Riddell 公式估算驻点热流，将其作为优化目标函数的惩罚项。
2.  **蒙特卡洛打靶**: 引入大气扰动和气动偏差，验证制导律的鲁棒性。
3.  **CUDA 加速**: 将气动插值和引力计算完全移至 GPU (目前仅引力部分有 CUDA 接口准备)。

## 4. 常用命令速查

**构建项目**:
```powershell
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

**运行单次仿真**:
```powershell
# 参数: [BoostStart] [BoostRate] [MinPitch] [TEnd] [GlideAoA] [PullUpAlt] [PullUpAoA]
.\bin\Release\AeroSim.exe 6.5 2.8 5.0 2000.0 12.0 40000.0 20.0
```

**运行自动优化**:
```powershell
cd e:\missile
python tune_parameters.py
```
